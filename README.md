# SAP Business One Core Implementation (IMRELEVSA)

This repository documents the foundational implementation of **SAP Business One** for IMRELEVSA. The project focused on transitioning the company's core operations into a centralized ERP environment, ensuring data integrity and process standardization across departments.

---

### 📋 Project Scope & Implementation

The implementation covered the end-to-end setup of the ERP system, focusing on creating a scalable architecture for industrial operations:

*   **Business Blueprinting:** Requirement gathering and mapping of industrial business processes to SAP B1 standard modules.
*   **Infrastructure Setup:** Installation and configuration of the server environment, including **IIS** optimization and SQL/HANA database connectivity.
*   **Functional Configuration:** Setup of core modules including Financials, Sales (A/R), Purchasing (A/P), and Inventory Management.
*   **System Hardening:** Troubleshooting critical server-side errors (such as HTTP 500 series in IIS) to ensure 24/7 system availability for the production environment.

### 🛠️ Technical Contributions

Even without full documentation, my role was pivotal in the **technical stabilization** of the project:
1.  **Server Migration:** Managed the deployment of system components across server environments.
2.  **Troubleshooting:** Identified and resolved complex handler mapping issues (Errors 500.19 and 500.21) that were blocking the communication layer.
3.  **Data Governance:** Established User-Defined Fields (UDFs) and validation rules to maintain high data quality during the go-live phase.
